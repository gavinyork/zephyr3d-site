# Tone mapping

Tone mapping is a process used to convert HDR images to LDR ones. Our tone mapping utilizes the ACES encoding system.

```javascript

// Enable tonemap
camera.toneMap = true;
// Exposure value
camea.toneMapExposure = 1.5;
```
